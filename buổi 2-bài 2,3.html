Plus, it introduces energy-saving features that extend battery life and reduce energy consumption. With Copilot+ PCs, you're not just keeping up with the times; you're setting the pace for what's possible in personal computing.

Built by listening to your feedback, this update also increases security to ensure a safer and more reliable experience.

Get early access to the latest features

To ensure you have access to the latest and greatest features of Windows, open Settings > Windows Update.

Here, select Check for updates.

And if you want to be among the first to get the latest non-security updates, set the toggle to On next to Get the latest updates as soon as they're available.

Are you a Windows Insider? Check out the Windows Insider blog to see what's new in the latest preview builds.
